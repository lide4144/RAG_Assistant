# Research Assistant Manual Acceptance

## 2-minute workflow checklist

- [ ] 在 `Library` 中一次导入 5 篇论文（可同一专题）。
- [ ] 导入反馈中明确显示成功/失败数量与下一步提示。
- [ ] 切换到 `Chat`，在默认界面直接看到提问输入框。
- [ ] 提出一个开放式科研问题并得到包含引用的回答。
- [ ] 点击引用后可在用户视图看到证据来源（论文/位置/片段）。
- [ ] 点击“生成灵感卡片”，跳转 `Ideas` 出现可编辑草稿。
- [ ] 编辑标题或研究问题并保存，确认保存内容为编辑后的文本。
- [ ] 在 `Ideas` 中将状态从 `draft` 推进到 `shortlisted`，更新时间发生变化。

## 回归检查

- [ ] 开启“显示开发者审查面板”后仍能查看 trace 与 evidence_grouped。
- [ ] `runs/*` 仍产出 `qa_report.json` 与 `run_trace.json`。
- [ ] `data/session_store.json` 仍可清空并保持会话隔离。

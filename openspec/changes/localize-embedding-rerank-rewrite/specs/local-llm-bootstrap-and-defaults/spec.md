## 新增需求

### 需求:系统必须提供本地模型安装与初始化基线
系统必须提供可复现的本地模型安装与初始化流程，并必须以 Ollama 作为默认主路径。流程必须覆盖 `embedding`、`rerank`、`rewrite` 三段模型的安装准备、启动校验与常见故障排查。

#### 场景:开发机按文档完成本地模型初始化
- **当** 开发者在 8GB 显存设备上按文档执行安装与初始化步骤
- **那么** 系统必须能够完成三段本地模型可用性检查，并输出明确的成功或失败信息

### 需求:系统必须定义本地模型默认档位与降级备选
系统必须内置本地模型默认值与降级备选：`embedding` 默认 `BAAI/bge-small-zh-v1.5`、备选 `BAAI/bge-base-zh-v1.5`；`rerank` 默认 `BAAI/bge-reranker-base`；`rewrite` 默认 `Qwen2.5-3B-Instruct` 4bit、备选 `Qwen2.5-1.5B-Instruct` 4bit。

#### 场景:默认档位不可用时切换备选
- **当** 默认模型拉取失败或运行不稳定
- **那么** 系统必须允许切换到对应备选模型并继续完成初始化流程

## 修改需求

### 需求:系统必须提供 LLM 连接设置面板
系统必须在“模型设置”页面同时提供“核心模型选择区”和“详细配置编辑区”。核心模型选择区必须允许用户直接切换回答模型、重排模型和问题改写模型，禁止仅以只读文本概览显示当前生效模型。

#### 场景:进入设置页切换核心模型
- **当** 管理员进入“模型设置”页面
- **那么** 页面必须提供可直接选择的核心模型下拉控件，而不是纯文本摘要

### 需求:系统必须提供模型连接设置的保存反馈
系统必须在“测试并保存”之外，为核心模型和导入增强配置提供更清晰的中文状态摘要。保存成功后，页面必须继续展示当前模型来源和导入增强状态，禁止只显示一次性提示而缺少稳定回显区域。

#### 场景:保存成功后查看概览
- **当** 管理员保存模型配置成功
- **那么** 页面必须在概览区显示当前模型、来源和导入增强摘要

## 修改需求

### 需求:系统必须提供 Marker 产物管理视图
系统必须将 Marker 产物管理视图中的主要中文文案改写为用户可理解表达，例如将 `Artifact Management` 表达为“生成文件管理”，将分组表达为“索引文件”“处理结果”。系统禁止在主界面标题中继续使用难懂的原始英文文件管理术语作为唯一表述。

#### 场景:查看产物总览
- **当** 管理员打开生成文件管理面板
- **那么** 页面必须以中文标题和分组名展示各类文件

### 需求:系统必须提供产物级操作与风险提示
系统必须继续支持复制路径、重建和删除操作，但按钮文案和健康提示必须改为中文友好表达，禁止在主操作区仅显示英文按钮。

#### 场景:查看单个产物卡片
- **当** 管理员浏览某个文件卡片
- **那么** 系统必须显示中文操作按钮和中文状态标签

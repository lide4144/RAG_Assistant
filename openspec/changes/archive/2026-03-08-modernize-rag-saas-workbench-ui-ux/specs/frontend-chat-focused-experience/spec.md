## 修改需求

### 需求:系统必须提供聊天空白状态
系统必须在无历史消息时展示居中快捷指令卡片组（至少 3 个），并必须支持用户点击卡片后直接将卡片文本作为输入发送，禁止出现无引导空白区域。

#### 场景:首次进入无对话
- **当** 当前会话没有任何消息
- **那么** 页面必须展示可点击的快捷指令卡片并支持一键发起提问

### 需求:系统必须在模型未配置时阻止误用并提供跳转入口
系统必须将聊天顶部模型摘要从拼接字符串重构为结构化标签组，至少包含“推理模型”“重写模型”等语义标签与对应模型值，禁止输出难以解析的斜杠串联文本。

#### 场景:展示模型摘要标签
- **当** 聊天页加载模型摘要信息
- **那么** 系统必须按语义标签分组展示模型信息并保持可读分隔

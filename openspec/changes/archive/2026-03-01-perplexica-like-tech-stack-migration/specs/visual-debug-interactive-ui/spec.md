## ADDED Requirements

### 需求:系统必须将开发者审查面板与产品界面解耦
系统必须允许在产品界面与开发者审查面板间独立开关；禁止默认向普通用户暴露调试字段。

#### 场景:普通用户模式隐藏审查字段
- **当** UI 运行在普通用户模式
- **那么** 页面必须隐藏 trace 原始字段，仅展示用户可理解信息

### 需求:系统必须支持 GraphRAG 调试可视化入口
开发者审查面板必须支持查看图扩展子图、路径与节点元数据，作为 `graph_expand` 标签的增强视图。

#### 场景:开发者审查中查看图路径
- **当** 当前轮存在图扩展证据
- **那么** 审查面板必须显示至少一条从 seed 到扩展节点的路径

## ADDED Requirements

### 需求:系统必须优先输出可追溯部分回答
当证据不足以支持完整回答但存在部分可追溯证据时，系统必须优先输出部分回答并明确不确定边界；禁止默认直接拒答。

#### 场景:部分证据可用时不直接拒答
- **当** 关键问题仅部分要素有证据支持
- **那么** 系统必须给出可追溯部分结论并明确列出缺失要素

### 需求:系统必须限制连续澄清打断
系统必须限制连续澄清轮次对话打断，在达到阈值后必须切换为“部分回答+最小澄清”策略。

#### 场景:连续澄清达到阈值
- **当** 用户连续多轮触发澄清
- **那么** 系统必须输出可追溯最小回答并仅附一个澄清问题

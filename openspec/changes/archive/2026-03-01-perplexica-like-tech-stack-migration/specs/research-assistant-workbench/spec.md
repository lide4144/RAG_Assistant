## ADDED Requirements

### 需求:系统必须提供产品化工作台入口
研究助手工作台必须支持产品化聊天入口与任务上下文延续，禁止仅依赖 Streamlit 单页作为唯一入口。

#### 场景:工作台从产品界面进入
- **当** 用户在新前端进入研究助手模式
- **那么** 系统必须复用历史会话上下文并提供连续追问能力

### 需求:系统必须保留研究流程可追溯性
工作台必须保留关键过程可追溯信息（query 演化、证据来源、决策原因），但必须以可折叠或按需展开方式呈现。

#### 场景:按需展开决策依据
- **当** 用户主动查看“为何这样回答”
- **那么** 系统必须展示该轮 query 演化与证据决策摘要

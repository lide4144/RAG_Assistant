## ADDED Requirements

### 需求:主题匹配分词必须支持中英混写边界
系统在执行主题匹配分词时，必须在字母/数字/中文边界处进行稳定切分；对于中英无空格混写问句（如 `Transformer是什么`），必须至少产生 `transformer` 与中文问句要素的可比较 token，禁止将整句固化为单一复合 token。

#### 场景:中英无空格混写可正确切分
- **当** 输入问题为 `Transformer是什么` 或等价中英无空格混写问句
- **那么** 主题匹配分词结果必须包含 `transformer`，且不得仅包含单个复合 token

### 需求:主题匹配分词必须在标点与全角符号下稳定
系统必须在中英文标点、全角符号或问号存在时保持分词稳定；同一语义问题在是否携带标点时，主题匹配分词不得出现结构性失真。

#### 场景:有无标点分词一致
- **当** 输入问题分别为 `Transformer是什么` 与 `Transformer是什么？`
- **那么** 两次分词都必须能与证据中的 `transformer` 形成有效匹配，且不应因标点差异导致 token 退化
